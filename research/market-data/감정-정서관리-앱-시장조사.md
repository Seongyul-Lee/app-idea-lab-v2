## 1. 한국 시장 주요 감정/정서 관리 앱 현황

### 1-1. 하루콩 (HaruBean) - 감정 기록 특화

| 항목 | 내용 |
|------|------|
| **개발사** | 블루시그넘 |
| **출시** | 2021년 |
| **누적 다운로드** | 글로벌 1,000만+ (2025년 기준) |
| **해외 비중** | 전체 사용자의 80% 이상 (미국 최다) |
| **서비스 국가** | 236개국 |
| **주요 사용층** | 10대 후반 ~ 30대 중반 (80~90%) |
| **수상** | 구글플레이 '올해의 앱' 2년 연속 선정 |
| **투자** | 누적 21억 원 유치 |

**주요 기능:**
- 콩 모양 이모티콘(16종)으로 감정 기록 (복수 선택 가능)
- 월별 캘린더로 감정 흐름 시각화
- 기록 블럭으로 활동/수면/운동/월경 등 추적
- 감정 인사이트 (활동-감정 상관관계 분석)
- 사진 + 한 줄 메모 기능
- 캐릭터 테마 스토어 (디즈니 콜라보 진행)

**수익 모델:**
- 구독 (광고 제거 + 프리미엄 기능)
- 캐릭터/테마 판매 (스토어)
- 무료 사용자 대상 광고

**사용자 불만:**
- iPad 미지원, 가로 모드 없음
- 기기 간 실시간 동기화 미지원 (수동 백업/복원 필요)
- 데이터 내보내기(PDF/사진) 기능 부재
- 타임라인 하단 일기가 네비게이션에 가려지는 UI 이슈
- 무료 사용 시 광고 노출

**강점과 약점 요약:**
- 강점: 직관적 UX, 귀여운 캐릭터, 간편한 기록 (몇 초 소요), 글로벌 확장성
- 약점: 감정 기록 '이후' 단계(분석/대응/관리) 기능 부족, 한국어 특화 감정 분석 미흡

### 1-2. 마인드카페 (MindCafe) - 심리상담 플랫폼

| 항목 | 내용 |
|------|------|
| **개발사** | (주)아토머스 (Atommerce) |
| **누적 사용자** | 400만+ |
| **포지셔닝** | No.1 온라인 익명 심리상담 플랫폼 |
| **오프라인** | 마인드카페 심리상담센터 운영 |

**주요 기능:**
- 24시간 전문 심리상담 (채팅/전화/화상/대면)
- 과학적 심리검사 (100문항, 무료) + 7일간 분석 결과
- 증상별 맞춤 솔루션 (우울/불안/불면/스트레스)
- 익명 커뮤니티 (고민 공유)
- EAP(기업 복리후생) 프로그램
- 감정 다이어리, 음성 감정 분석

**수익 모델:**
- 상담 비용 (유료 전문가 매칭)
- EAP 기업 계약
- 프리미엄 서비스

**사용자 불만:**
- 앱 오류/크래시 발생
- 해외 결제 불가 (해외 신용카드, PayPal, Venmo 미지원)
- 상담 비용 부담 (전문가 상담은 유료)

**강점과 약점 요약:**
- 강점: 전문 상담사 검증 시스템, 24시간 접근성, 익명성, 다양한 상담 채널
- 약점: '감정 기록' 보다 '상담' 중심, 가벼운 일상 감정 관리에는 과도함

| 항목 | 내용 |
|------|------|

| **고객 수** | 80만+ |

- AI 챗봇 '티티' (AI 셀프 상담)
- 명상/ASMR 사운드테라피
- 감정일기 + 마음관리 루틴
- 커뮤니티 (MBTI별 등 주제별)
- 상담사 필터링 (성별, 특수 경험: 해외거주/이혼/퀴어프렌들리/성폭력 등)
- 정신과 병원/약물 정보 제공

- 심리상담 비용 (3~4만원대/회, 오프라인 대비 저렴)
- 프리미엄 콘텐츠

**사용자 불만:**
- 결제/로그인/상담방 오류 빈번
- 해외 사용 시 채팅 업로드 속도 저하
- 앱 안정성 이슈 반복

**강점과 약점 요약:**
- 강점: AI 챗봇 + 전문가 상담 연계, 명상/ASMR 콘텐츠, 상담사 필터 상세
- 약점: 기술적 안정성 문제, '감정 기록/추적' 자체의 깊이 부족

| **한국어 감정 어휘 특화** | 한국어 감정 표현의 미묘한 차이(서운하다, 답답하다, 억울하다 등)를 반영하는 앱 부재 |

| **감정 대응 가이드** | 기록 후 "그래서 어떻게 해야 하는가?"에 대한 실행 가능한 가이드 부족 |
| **비전문가 수준의 셀프케어** | 전문 상담까지는 필요 없지만, 단순 기록보다는 더 능동적인 관리가 필요한 중간층 |
| **프라이버시 중심 설계** | 감정 데이터의 민감성 대비, 온디바이스 처리/완전 익명화 강조하는 서비스 부족 |

## 3. 2026 '필코노미(Feelconomy)' 트렌드와의 연관성

### 3-1. 필코노미란?

- **출처**: 『트렌드 코리아 2026』 (김난도 서울대 명예교수)
- **정의**: 감정(Feel) + 경제(Economy). 소비자의 정서와 공감이 구매 결정의 중심에 놓이는 현상
- **핵심 메시지**: "기분이 돈이 된다" - 소비자의 기분을 살피고 배려하는 기업과 서비스가 성공

### 3-2. 필코노미의 핵심 특징

| 특징 | 설명 |
|------|------|
| 감정이 소비의 직접적 동기 | 필요가 아닌 감정 반응으로서의 소비 |
| 기분 관리로서의 소비 | 감정은 '관리해야 할 프로젝트' |
| 일상 소비재에서 두드러짐 | 고가 명품이 아닌 일상 영역에서 감성적 요소 강화 |
| MZ세대 중심 | "이 브랜드가 나에게 어떤 감정을 주는가"로 판단 |

### 3-3. 감정/정서 관리 앱과의 연결점

1. **감정 인식의 일상화**: 필코노미 시대 소비자는 자신의 감정 상태를 '인식 → 진단 → 관리 → 전환'하는 루틴을 원함. 감정 기록 앱은 이 첫 단계를 담당
2. **감정 데이터의 상업적 가치**: 사용자의 감정 패턴 데이터는 맞춤형 콘텐츠/제품 추천의 핵심 자산
3. **기분 전환 서비스 수요**: 단순 기록을 넘어, 부정적 감정을 전환시키는 '행동 제안'의 가치 증대
4. **정서적 경험 디자인**: 앱 자체의 UX가 정서적 만족을 줘야 함 (하루콩의 콩 캐릭터가 좋은 예시)

### 3-4. 메타센싱(2026 Z세대 트렌드)

- **출처**: 대학내일20대연구소
- **정의**: 감정 관리가 하나의 핵심 '역량'이 되는 시대
- Z세대는 MBTI/TCI처럼 감정도 객관적으로 분석하고 싶어함
- 부정적 감정을 '기분 탓'으로 치부하지 않고, 원인을 파악하려는 욕구
- 생성형 AI 활용 20대 중 24.5%가 AI에게 심리 상담 경험 있음

## 4. 한국어 특화 감정 기록 앱이 부족한 이유 분석

### 4-1. 시장 구조적 요인

| 요인 | 설명 |
|------|------|
| **시장 규모 인식** | 한국 단일 시장만으로는 감정 기록 앱의 수익화가 어렵다는 인식. 하루콩도 해외 사용자 80%에 의존 |
| **글로벌 앱의 한국어 추가 방식** | Daylio 등 글로벌 앱이 30개+ 언어로 로컬라이즈. 한국어 '번역'은 되지만 한국 감정 문화 반영 부족 |
| **멘탈헬스 스티그마** | 한국 사회에서 정신건강 앱 사용에 대한 사회적 낙인이 여전히 존재. 시장 진입 장벽으로 작용 |
| **수익화 난이도** | 감정 기록만으로는 유료 전환율이 낮음. 상담 연계나 프리미엄 분석 기능이 필요하나 1인 개발로 구현 난이도 높음 |

### 4-2. 기술적 요인

| 요인 | 설명 |
|------|------|
| **한국어 감정 NLP 난이도** | 한국어 감정 표현은 맥락 의존성이 높고, 미묘한 뉘앙스 차이가 큼 (서운하다 vs 섭섭하다 vs 아쉽다) |
| **감정 분류 체계 부재** | 영어권 감정 분류(Plutchik's Wheel 등)를 한국 문화에 맞게 재설계한 표준이 없음 |
| **데이터 부족** | 한국어 감정 라벨링 데이터셋이 영어 대비 절대적으로 부족 |

### 4-3. 문화적 요인

| 요인 | 설명 |
|------|------|
| **한국 고유 감정 표현** | '정', '한', '눈치', '답답하다', '억울하다' 등 영어로 직역 불가한 감정 개념이 다수. 기존 앱은 이를 반영하지 못함 |
| **감정 표현 문화** | 한국 문화에서 감정을 직접적으로 표현하는 것에 대한 문화적 익숙하지 않음. 앱이 이 장벽을 낮춰야 함 |
| **기록 방식 선호** | 한국 사용자는 긴 글쓰기보다 간편한 아이콘/이모티콘 기반 기록을 선호 (하루콩의 성공 요인) |

### 4-4. 경쟁 환경 요인

| 요인 | 설명 |
|------|------|
| **상담 앱과의 혼재** | 마인드카페/트로스트가 '감정 관리' 카테고리를 점유하고 있으나, 실제로는 '심리상담' 서비스. 순수 감정 기록/관리 앱과 포지셔닝 혼란 |
| **하루콩의 글로벌화** | 한국 대표 감정 기록 앱이 글로벌 전략을 취하면서, 한국어 특화보다 범용성에 집중 |

## 7. 시장 기회 요약

### 7-1. 미충족 수요 (Unmet Needs)

1. **"기록과 관리 사이"의 공백**: 하루콩처럼 간편하게 기록하되, 트로스트처럼 능동적으로 관리까지 해주는 중간 포지션
2. **한국어/한국 문화 특화 감정 체계**: '서운하다', '답답하다', '눈치 보인다' 등 한국 고유 감정 표현을 인식하고 분류
3. **비전문가 레벨의 셀프 케어**: 전문 상담(3~10만원/회)은 부담스럽지만, 단순 기록은 부족한 사용자층
4. **AI 기반 감정 패턴 분석**: 감정-활동-시간-장소 상관관계를 AI로 분석하고 실행 가능한 제안 제공
5. **프라이버시 우선 설계**: 감정 데이터의 민감성을 고려한 온디바이스 처리, 완전 익명화

### 7-2. 필코노미 시대의 핵심 가치 제안

- "기분을 기록하는 것"에서 "기분을 관리하는 것"으로의 전환
- 감정 인식 → 패턴 발견 → 대응 행동 제안의 완결된 루프
- 한국 사용자의 감정 표현 방식에 최적화된 UX
