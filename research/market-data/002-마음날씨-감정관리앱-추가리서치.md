### 1-2. 핵심 기능

**무료 기능 (기본):**
- 루틴/할일 관리 (요일별, 카테고리별 설정)
- TODO 날짜/시각/알람 설정
- 아기 루빗(토끼 캐릭터) 육성 - 루틴 완료 시 성장
- 당근 보상으로 루빗의 방 꾸미기
- 다양한 알람 (기분 좋은 알람~시끄러운 알람)
- 기본 감정 기록 기능
- 루틴 최대 8개 (무료 제한)

**프리미엄 기능 (유료 구독):**
- 무제한 루틴 생성
- AI 위로의 편지 (ChatGPT 기반) - 감정 일기 분석 후 맞춤 위로 편지 전달
- AI 습관코치 챗봇 '에디' (루틴 박사 토끼 캐릭터) - 습관 기록 기반 조언, 습관 추천, 응원/위로 채팅
- NFC 태그 기반 강제 기록 기능
- 추가 캐릭터/테마

### 1-4. 사용자 리뷰 분석

**긍정적 리뷰:**
- "절대적으로 사랑스러운 앱, 아트가 매우 귀엽다"
- "일주일 사용 후 큰 변화. 재택근무 시 가사, 건강, 셀프케어, 업무 루틴 구축에 도움"
- "쉽고 간편한 인터페이스, 오늘의 할 일과 루틴을 꾸준히 실천할 수 있도록 가이드"
- ADHD/우울증 사용자들에게 특히 유용하다는 평가

**부정적 리뷰:**
- "이론적으로 귀엽고 유익하지만, 투박하고 간소화되지 않았다"
- "아침 루틴을 과제별로 하나씩 추가해야 하는 것이 불편"
- 무료 버전 8개 루틴 제한에 대한 불만
- 앱 렉, 데이터 소모가 크다는 의견
- 구독 가격($40/년)에 대해 "무제한 루틴만 원하는데 너무 비싸다" - 티어별 구독 요청

### 1-6. 마음날씨와의 비교 시사점

| 비교 항목 | 루빗 | 마음날씨 (기획) |
|----------|------|----------------|
| **핵심 축** | 습관/루틴 관리가 메인, 감정은 부가 기능 | 감정 기록/관리가 메인, 행동 제안은 감정 기반 |
| **AI 활용** | ChatGPT 기반 위로 편지/습관 코치 | 감정 패턴 분석 + 맞춤형 셀프케어 행동 제안 |
| **감정 체계** | 범용적 감정 기록 | 한국어 감정 어휘 특화 |
| **타겟** | 자기계발/루틴 관리 니즈 사용자 | 감정 관리/셀프케어 니즈 사용자 |
| **게이미피케이션** | 강함 (캐릭터 육성, 보상) | 미정 (가볍게 적용 가능) |

**핵심 시사점:** 루빗은 "습관 관리 앱에 감정 기능을 추가"한 형태이고, 마음날씨는 "감정 관리 앱에서 행동 제안으로 연결"하는 형태로 방향이 다름. 직접 경쟁보다는 인접 시장에 위치.

## 2. 데일리오(Daylio) 한국 시장 현황

### 2-1. 기본 정보

| 항목 | 내용 |
|------|------|
| **개발사** | Daylio (슬로바키아) |

| **최신 버전** | 1.63.12 (2025년 12월 24일) |
| **포지셔닝** | 글로벌 1위 무드 트래커 & 일기 앱 |
| **App Store 설명** | "기분 추적과 감정 기록을 위한 감성 일기 앱" |

### 2-2. 가격 정보

| 구독 유형 | 가격 |
|----------|------|
| **월간 구독** | $2.99/월 |
| **연간 구독** | $23.99/년 |
| **프리미엄 잠금 해제** | 추가 기분(mood), 알림, 고급 통계 |

### 2-3. 주요 기능 (2025-2026 최신)

**기본 기능:**
- 기분 선택 + 활동 추가 (타이핑 없이 기록 가능)
- 메모, 사진, 음성 메모 추가
- 주간/월간/연간 차트 통계
- 'Year in Pixels' 연간 보기
- 일간/주간/월간 목표 설정
- 습관 형성 및 업적 수집
- Google Drive 백업/복원
- PIN 잠금, PDF/CSV 내보내기

**최신 업데이트 기능:**
- **스케일(Scales) 기능**: 수면, 스트레스, 에너지, 통증 등을 슬라이더로 추적 (2025년 신규)
- **활동 위젯**: 홈 화면에서 습관 반복 횟수 확인
- **햅틱 피드백**: 기분 기록 시 촉각적 피드백
- **활동 색상 커스터마이징**: 통계에서 색상 균형 확인
- 고급 통계에서 각 색상별 세부 정보 확인

### 2-4. 한국 사용자 반응

**긍정적 반응:**
- "감성적인 면보다 기능적인 면에서 강점" - 딱딱 쓰고 싶은 것만 쓰는 사용자에게 적합
- 2026년 디지털 다이어리 추천에서 "감정 기록이 필요하면 Daylio 같은 기분 트래커형이 가볍고 빠르다"는 평가

**부정적 반응 / 한계:**
- 한국적 "감성" 부족 - 번역은 되어 있지만 한국 감정 문화 반영 미흡
- 한국어 감정 어휘의 미묘한 차이(서운하다, 답답하다, 억울하다 등) 미반영
- AI 기반 감정 분석이나 맞춤형 제안 기능 없음 (데이터 시각화에 그침)

### 2-5. 마음날씨 대비 Daylio의 약점 (진입 기회)

| 영역 | Daylio의 한계 | 마음날씨 기회 |
|------|-------------|-------------|
| **감정 체계** | 범용 감정 5단계 (최고~최악) | 한국어 고유 감정 어휘 반영 |
| **AI 분석** | 통계/차트만 제공, AI 분석 없음 | AI 패턴 분석 + 행동 제안 |
| **문화적 맞춤** | 30개+ 언어 번역본, 한국 문화 미반영 | 한국 사용자 특화 UX |
| **액션 연결** | 기록 후 분석까지만 (수동적) | 기록 → 분석 → 행동 제안 (능동적) |
| **가격** | $23.99/년 (~32,000원) | 경쟁력 있는 가격 설정 가능 |

### 4-5. 주 사용 시간대 관련 동향

> 구체적인 감정 관리 앱 시간대별 이용 통계는 공개 자료에서 확인되지 않음. 다만 관련 간접 지표:

- 하루콩은 "하루 마무리" 시점에 기록하는 패턴이 주를 이룸 (저녁~취침 전)
- 루틴 관리 앱은 아침(기상 직후)과 저녁(취침 전)에 사용 피크
- 명상 앱(마보, 코끼리)은 취침 전 시간대에 사용 집중

### 4-6. 셀프케어 메가 트렌드 (2025)

- 셀프케어가 "메가 트렌드"로 자리잡음 - 뷰티/신체건강을 넘어 **정신건강**이 중요 분야로 부상
- 2030세대 자기관리 동기: "나다움 찾기" (남에게 잘 보이기가 아닌 자기만족)
- HLY(Healthy Life Years, 건강수명) 개념 부상 - 단순 장수가 아닌 건강한 활동 수명
- 2025년 MZ세대 핵심 키워드: **"불안 관리"** - 고금리, 경기둔화, AI 일자리 불안으로 통제 가능한 영역에 집중
- 정신건강 관리 관련 '얼리케어 신드롬' - 치료보다 **예방** 중심 소비

### 7-2. 마음날씨의 차별화 포인트

1. **한국어 감정 어휘 특화**: 서운하다/섭섭하다/아쉽다 등 미묘한 뉘앙스 차이를 인식하는 앱은 시장에 부재
2. **기록 -> 분석 -> 행동 제안 완결 루프**: 하루콩(기록만), Daylio(기록+통계), 트로스트(상담) 사이의 공백 포지션
3. **비전문가 레벨 셀프케어**: 전문 상담(3~10만원/회)은 부담, 단순 기록은 부족한 중간 사용자층
4. **AI 기반이지만 가벼운 접근**: 라일리 하루(기술 중심)와 달리, 사용자 친화적이고 따뜻한 UX
