# 004-소음일지 시장분석

## 조사일: 2026-01-31

---

## 1. 시장 규모 추정 (TAM → SAM → SOM)

### TAM (Total Addressable Market) - 전체 시장
- **한국 공동주택 거주 가구**: 약 1,200만 가구 (국토교통부 공동주택 현황 기준, 아파트·연립·다세대 포함)
- **층간소음 경험률**: 78% (환경부/네이버 블로그 인용 데이터)
- **층간소음 경험 가구**: 약 936만 가구
- **TAM 추정**: 936만 가구 × 월 3,900원(기본 구독) × 12개월 = **약 4,380억 원/년**

### SAM (Serviceable Addressable Market) - 접근 가능 시장
- **적극적으로 층간소음 해결을 원하는 가구 비율**: 10~15% 추정
  - 근거: 환경공단 이웃사이센터 연간 전화상담 3만~4만 건, 이는 실제 불만 가구의 극히 일부
  - 네이버 지식iN "층간소음 증거" 관련 질문 3,300건 이상
  - 층간소음 커뮤니티(카페 "층간소음과 피해자 쉼터") 활발한 활동
- **SAM 추정**: 936만 × 10% = 약 93.6만 가구
- **SAM 금액**: 93.6만 × 월 3,900원 × 12개월 = **약 438억 원/년**

### SOM (Serviceable Obtainable Market) - 실제 획득 가능 시장
- **1인 개발 앱의 현실적 침투율**: SAM의 1~3% (1~3년 내)
- **SOM 추정**: 93.6만 × 2% = 약 1.87만 유료 사용자
- **SOM 금액**: 1.87만 × 월 3,900원 × 12개월 = **약 8.7억 원/년**
- **일회성 리포트 구매 추가 수익**: 무료 사용자 중 5%가 연 2회 리포트 구매 시 약 1~2억 원 추가

### 산출 논거
1. 환경공단 층간소음 이웃사이센터 전화상담: 2024년 33,027건, 2020~2025년 7월 누적 21만 7,387건 (하루 평균 106.6건)
2. 층간소음 기준 초과 건수 증가 추세: 2020년 18건 → 2024년 88건 (매년 증가)
3. 층간소음 원인 67.6%가 '뛰거나 걷는 소리'로 가장 빈번한 일상적 소음
4. 2012년 8,795건 → 2022년 40,393건으로 10년간 359% 급증 (2024년에도 3만 3천건+)

## 2. 시장 트렌드 및 성장 동인

### 성장 동인
1. **사회적 이슈화**: 층간소음으로 인한 강력범죄(살인, 폭행) 뉴스 반복 보도 → 사회적 인식 강화
2. **법적 대응 증가**: 위자료 300만 원 인정 판례(2025년 서울남부지법), 환경분쟁조정 활용 증가
3. **정부 기준 강화**: 층간소음 인정 기준 하향 (주간 39dB, 야간 34dB로 4dB 하향)
4. **1~2인 가구 증가**: 소음에 민감한 자취 청년·1인 가구 확대
5. **재건축/리모델링 증가**: 인테리어 공사 소음 민원도 사회적 이슈

### 시장 환경
- 데시벨 측정 앱은 이미 대중화 (구글플레이 "소음측정기" 1,000만+ 다운로드)
- 그러나 **증거 수집 특화 앱**은 시장에 공백 존재
- 변호사·법무법인 블로그에서 공통적으로 "소음일지 + 측정 + 녹음 + 진단서 조합" 권고
- DL이앤씨 등 건설사들이 스마트홈에 층간소음 측정·알림 기능 내장 시작 (잠재적 위협이자 시장 검증)

## 3. 고객 직무(JTBD) 분석

### 핵심 직무
> "층간소음으로 인한 피해를 객관적으로 증명하고, 이를 기반으로 실질적 해결(관리사무소 민원, 환경분쟁조정, 법적 대응)을 이끌어내고 싶다"

### 기존 대안의 부족 충족 지점
| 기존 대안 | 부족 충족 지점 |
|-----------|----------------|
| 스마트폰 데시벨 앱 (소음측정기 등) | 단순 실시간 측정만 제공. 자동 감지, 이력 관리, 소음일지, PDF 리포트 없음. 법적 증거 가치 낮음 |
| 수기 소음일지 | 노동 집약적, 비체계적, 데시벨 수치 부재 |
| 이웃사이센터 측정 | 신청 후 방문까지 시간 소요, 야간/불규칙 소음은 측정 어려움, 윗집 동의 필요한 경우 있음 |
| 전문 소음측정기(제로리빙 등) | 3~5만원 하드웨어 별도 구매, 앱 연동 불가, 증거 리포트 자동 생성 미지원 |

### 네이버 블로그/카페/지식iN 실제 불만 근거

**지식iN 대표 질문들:**
- "층간소음 증거 어떻게 모으나요?" - 관련 질문 3,300건+
- "층간소음 녹음 파일로 충분할까요?" - 변호사들이 일관되게 "녹음 + 소음일지 + 공식 측정 + 진단서 조합 필요" 답변
- "소음측정어플 작동중인 휴대폰을 촬영한 동영상" - 증거 수집의 번거로움 호소

**카페 "층간소음과 피해자 쉼터" 대표 글들:**
- "데시벨 측정 어떻게 하시나요? 매번 동영상 찍어대니 용량 부족"
- "이웃사이센터 측정 요청했는데 윗집이 비우는 날이면 못 잡아"
- "3년 당하다가 층간소음 영상기록앱 만든 사연" - 개인 개발자가 직접 만들 정도로 수요 존재
- "휴대폰으로 층간소음 못 찍는 이유" - 스마트폰 마이크 한계에 대한 실험 게시물

**법률 블로그 공통 권고:**
- "소음일지 + 녹음 원본 + 데시벨 측정 + 공식 측정 결과 → 조합 제출이 효과적"
- "한 달간 쌓인 소음일지는 '소음의 주기성'을 증명" (뉴로직 층간소음 연구소 블로그)

## 4. 수익화 가능성

### 유사 앱 과금 모델 참고
| 앱 | 과금 방식 | 비고 |
|----|-----------|------|
| 소음측정기 (Sound Meter) | 무료 + 광고 | 1,000만+ 다운로드, 광고 기반, 프리미엄 기능 없음 |
| Decibel X | 프리미엄 구독 $3.99/월 | 전문가 수준 측정, CSV 내보내기 등 유료 |
| 층소기 | 무료 기본 + 유료 기능 | 영상+데시벨 동시 기록, PDF 내보내기 |
| The Noise App (영국) | 무료 (주거 제공자 협약) | B2B 모델, 한국 미진출 |

### 사용자 지불 의향 판단
- **강점**: 층간소음은 감정적 고통이 크고, 법적 대응(위자료 100~300만 원 수준)을 위한 투자 의향 높음
- **월 3,900~5,900원 구독**: 위자료 배상(수십~수백만 원) 대비 저렴한 투자로 인식될 가능성
- **일회성 리포트 1,900원**: 진입 장벽 낮은 과금 → 전환율 높을 것으로 예상
- **리스크**: 무료 데시벨 앱이 많아 "증거 품질 차이"를 충분히 설득해야 유료 전환 가능
