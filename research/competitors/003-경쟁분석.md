# 003-살았니 (SafePin) 경쟁 분석

## 조사일: 2026-01-31

---

## 1. 직접 경쟁자 분석

### 1-1. Demumu (死了麼 / "Are You Dead?") — 중국
- **출시**: 2025년 6월 (iOS), 2026년 1월 바이럴
- **개발**: Yuejing (Zhengzhou) Technology, 1990년대생 3인 팀 원격 협업
- **기능**: 매일 체크인 버튼 → 2일 연속 미체크인 시 긴급연락처에 이메일 알림
- **가격**: 초기 무료 → 1위안 → 8위안(~1,500원) 유료 다운로드
- **성과**: 중국 Apple App Store 유료 앱 1위, 미국/싱가포르/홍콩 유틸리티 차트 1~2위, 10만+ 다운로드
- **강점**: 극도로 단순한 UX, 바이럴 네이밍, 글로벌 언론 보도
- **약점**:
  - 이메일 기반 알림만 지원 → 한국에서 이메일 알림은 비실용적 (카카오톡/SMS 문화)
  - SMS/전화 알림 미지원 (중국 비평에서도 지적됨: "죽음 통보 도구에 불과")
  - 체크인 주기 48시간 고정 → 커스터마이징 불가
  - 2026년 1월 15일 중국 앱스토어에서 제거됨
  - 한국어 미지원, 한국 시장 직접 진출 계획 미확인
  - "Demumu"로 글로벌 리브랜딩 중이나 한국 로컬라이징 없음

### 1-2. Snug Safety — 미국
- **출시**: 약 3년 이상 운영 중
- **기능**: 매일 체크인 → 미체크인 시 긴급연락처에 문자/전화 알림
- **가격**: 기본 무료
- **성과**: 2,000만+ 체크인 달성, AARP·Forbes 추천, Reddit AMA에서 호평
- **강점**: 안정적 서비스, 미국 시니어 시장 침투, 무료 모델로 사용자 기반 확보
- **약점**:
  - 영어 전용, 한국 미진출
  - 한국식 알림 채널(카카오톡) 미지원
  - 위치 추적 기반 → 프라이버시 우려
  - 수익 모델 불분명 (기본 무료, 프리미엄 전환 비율 미공개)
  - 한국 시장 진출 계획 없음

### 1-3. 루키스 "똑똑안부확인 서비스" — 한국
- **출시**: 2020년대 초반부터 운영
- **기능**: 생활 데이터(수발신 이력, 걸음 수) 자동 모니터링 → 일정 시간 변화 없을 시 AI 안부확인 전화 자동 발신
- **대상**: B2G(지자체 납품), 고령층/취약계층 한정
- **가격**: 지자체 예산으로 운영 (개인 사용자 직접 구매 불가)
- **강점**: 한국 시장 운영 경험, 지자체 네트워크, IoT 연동
- **약점**:
  - B2C 서비스 아님 — 일반 청장년은 이용 불가
  - 고령층 특화 UX — 20~50대 타겟 부적합
  - 지자체 사업 의존 → 확장성 제한
  - 능동적 셀프 체크인이 아닌 수동 모니터링 방식

## 2. 간접 경쟁자 / 대체재

| 서비스 | 유형 | 한계 |
|--------|------|------|
| 서울 안심이 앱 | 귀가 안전 (경로 추적, CCTV 연계) | 일일 체크인 기능 없음, 귀가 중심 |
| 보건복지부 복지위기알림 | 전력/통신 빅데이터 수동 모니터링 | 능동적 셀프 체크인 아님, 사후 대응 |
| 에이나인 돌봄플러그 | IoT 스마트플러그 (전력 사용 감지) | 별도 하드웨어 필요, 고령층 대상, B2G |
| Apple Watch 낙상 감지 | 웨어러블 자동 감지 | 고가 디바이스 필요, 낙상만 감지 |
| 가족/친구 수동 연락 | 비기술적 대안 | 매일 연락 부담, 체계적이지 않음 |

## 3. 진입장벽 평가

### 기존 앱의 네트워크 효과
- **낮음**: 안부확인 앱은 소셜 네트워크 효과가 약함. 사용자-긴급연락처(1:N, N≤3) 관계로 바이럴 계수가 제한적.
- 다만 긴급연락처 등록 시 상대방도 앱 설치 유도 가능 → 약한 수준의 바이럴 루프 존재.

### 전환 비용
- **매우 낮음**: 체크인 앱 특성상 데이터 락인이 거의 없음. 체크인 기록/스트릭 정도가 전환 비용.
- Demumu 사용자가 한국어 서비스로 전환하는 것은 매우 쉬움.

### 데이터 해자
- **없음**: 체크인 기록은 개인 데이터일 뿐 네트워크 가치를 생성하지 않음.
- 장기적으로 건강 데이터/패턴 분석이 차별화 포인트가 될 수 있으나 MVP 단계에서는 해당 없음.

### 대기업 위협
- **중간**: 카카오(카카오톡 기반 안부확인), 네이버(라인 기반)가 유사 기능을 추가할 가능성 존재.
- 그러나 안부확인은 메신저의 핵심 기능이 아니며, 별도 앱으로 출시할 인센티브가 낮음.
- 통신사(SKT, KT) AI 안부확인은 B2G 중심으로 B2C 확장 가능성은 있으나, 청장년 타겟은 아직 미착수.

## 4. 경쟁 우위 / 차별화 전략

### 살았니(SafePin)의 차별화 포인트

| 차별점 | 설명 |
|--------|------|
| **한국어 네이티브** | 한국 유일의 청장년 대상 안부확인 앱 (경쟁자 전무) |
| **카카오톡 알림** | 한국 사용자 95%+ 사용. 이메일 알림(Demumu)과 결정적 차이 |
| **체크인 주기 커스터마이징** | 12시간/24시간/48시간 선택 가능 (Demumu는 48시간 고정) |
| **SMS 이중 알림** | 카카오톡 + SMS 이중 발송으로 알림 도달률 극대화 |
| **20~50대 타겟** | 기존 정부 서비스(고령층)와 비경쟁 포지셔닝 |
| **안부 타임라인** | 체크인 스트릭, 한줄 메모 — 습관화 + 정서적 기록 |

### 위험 요소
1. **Demumu 한국 진출**: 글로벌 확장 중이므로 한국어 지원 추가 가능성 있음. 선점 시간이 핵심.
2. **카카오/네이버 진입**: 메신저 내 안부확인 기능 추가 가능성. 다만 별도 앱 출시 가능성은 낮음.
3. **무료 서비스 압박**: Snug Safety(무료), 정부 서비스(무료)와의 가격 경쟁.
4. **카피캣**: Demumu 바이럴 후 이미 미국에서 유사 앱 등장 (Wikipedia 기술). 한국에서도 유사 시도 가능.
