# 004-소음일지 경쟁분석

## 조사일: 2026-01-31

---

## 1. 직접 경쟁자 분석

### 경쟁자 1: 층소기 (층간소음측정/기록 앱)
- **플랫폼**: Android (구글플레이)
- **핵심 기능**: 영상 촬영 + 데시벨 동시 측정, 날짜/시간/GPS 자동 기록, PDF 내보내기, 캘린더 형식 통계
- **개발자**: 개인 개발자 (층간소음 3년 피해 후 직접 개발)
- **가격**: 무료 기본 + 유료 기능 (상세 미확인)
- **강점**:
  - 한국 층간소음에 특화된 유일한 앱
  - 영상+데시벨 동시 기록은 독보적 기능
  - PDF 변환 기능으로 증거 제출 편의성
  - 층간소음 피해자 커뮤니티(카페)에서 인지도 보유
- **약점**:
  - 백그라운드 자동 감지 미지원 (수동 기록만)
  - 한국 환경부 법적 기준 대비 자동 판단 기능 없음
  - 소음 패턴 분석(요일/시간대 통계) 부족
  - 24시간 연속 모니터링 불가
  - iOS 미지원 (Android only)

### 경쟁자 2: 소음측정기 (Sound Meter by rootApps)
- **플랫폼**: Android
- **다운로드**: 1,000만+ / 평점 4.7 / 리뷰 18.4만개
- **핵심 기능**: 실시간 데시벨 게이지, 최소/최대/평균 표시, 그래프, 보정 기능
- **개발자**: 개인 개발자 (서울 도봉구)
- **가격**: 무료 + 광고 + 인앱 구매
- **강점**:
  - 압도적 다운로드 수 (한국 1위 소음측정 앱)
  - 깔끔한 UI, 높은 평점
  - 기기별 보정 기능
- **약점**:
  - 단순 실시간 측정만 제공 — "참고 수준"이라고 앱 설명에 명시
  - 소음일지/기록 관리 기능 없음
  - 녹음 기능 없음
  - PDF 리포트 없음
  - 법적 기준 대비 판단 없음
  - 백그라운드 자동 감지 없음

### 경쟁자 3: Decibel X (SkyPaw Co.)
- **플랫폼**: iOS, Android
- **핵심 기능**: 전문가 수준 dB 측정, 주파수 분석, 히스토리, CSV 내보내기
- **가격**: 무료 기본 / 프리미엄 구독 $3.99/월
- **강점**:
  - 측정 정확도 높음 (NIOSH/OSHA 기준 지원)
  - 크로스플랫폼 (iOS + Android)
  - CSV 데이터 내보내기
- **약점**:
  - 한국어 미지원 / 한국 법적 기준 미내장
  - 직업 안전 보건 측정 목적 → 층간소음 특화 아님
  - 소음일지, 증거 리포트, 자동 감지 없음
  - 한국 사용자 커뮤니티/인지도 미미

## 2. 간접 경쟁자 / 대체재

| 대체재 | 설명 | 한계 |
|--------|------|------|
| 수기 소음일지 | 변호사 권고 방식. 노트/엑셀에 직접 기록 | 노동 집약적, 데시벨 수치 없음, 체계성 부족 |
| 이웃사이센터 측정 서비스 | 정부 무료 서비스. 전문 장비 방문 측정 | 신청→방문 시차, 야간/불규칙 소음 대응 어려움, 윗집 거부 가능 |
| 하드웨어 소음측정기 (제로리빙 등) | 전문 측정기 3~5만원 | 자동 기록 없음, 앱 연동 제한, 소음일지/리포트 미지원 |
| CCTV/홈카메라 녹화 | 소리+영상 동시 기록 가능 | 전용 장비 필요, 소음 패턴 분석 없음, 증거 리포트 수동 제작 |

## 3. 차별점 분석

### 소음일지 앱의 핵심 차별점

| 기능 | 소음일지 (우리) | 층소기 | 소음측정기 | Decibel X |
|------|----------------|--------|-----------|-----------|
| 백그라운드 자동 감지 | O (핵심) | X | X | X |
| 한국 법적 기준 내장 | O (환경부 기준) | X | X | X (NIOSH) |
| 소음일지 자동 생성 | O | 수동 기록 | X | X |
| PDF 증거 리포트 | O (법적 기준 대비표 포함) | O (기본) | X | X |
| 소음 패턴 분석 | O (요일/시간대) | 캘린더 기본 | X | 히스토리 |
| 녹음 파일 보존 | O (타임스탬프 보존) | O (영상) | X | X |
| iOS + Android | O | Android only | Android | O |

### 핵심 차별화 포인트
1. **"자동"**: 수동 측정 → 24시간 백그라운드 자동 감지. 야간/새벽 소음도 놓치지 않음
2. **"법적 기준"**: 한국 환경부 층간소음 기준을 내장하여 초과 여부 자동 판단
3. **"원스톱"**: 감지 → 기록 → 소음일지 → 증거 리포트까지 하나의 앱에서 완결

## 4. 진입장벽 및 위협 평가

### 진입장벽
- **네트워크 효과**: 없음 (개인 도구 앱이므로 네트워크 효과 불필요)
- **전환 비용**: 낮음 (축적된 소음 데이터가 있으면 일부 전환 비용 발생)
- **데이터 해자**: 개인별 소음 기록 데이터 축적 → 시간이 지날수록 전환 비용 증가
- **기술적 해자**: 백그라운드 오디오 처리 + 배터리 최적화 기술은 모방 가능하나 최적화에 노하우 필요

### 대기업/기존 서비스 위협
- **건설사 스마트홈 (DL이앤씨 등)**: 월패드에 층간소음 측정·알림 기능 탑재 시작. 그러나 신축 아파트에 한정, 증거 리포트 기능 없음
- **네이버/카카오**: 층간소음 전용 앱 출시 가능성 낮음 (니치 시장, 수익성 미미)
- **이웃사이센터 앱화**: 정부가 공식 앱을 만들 가능성 있으나, 정부 앱의 UX 한계와 느린 개발 속도로 단기 위협 낮음

### 종합 위협도: **낮음~중간**
- 대기업이 진출하기엔 시장이 니치
- 기존 경쟁 앱들은 "자동 감지 + 법적 기준 + 증거 리포트" 조합을 갖추지 못함
- 가장 큰 위협은 층소기 앱의 기능 확장이나, 현재 Android only + 수동 기록 한계
