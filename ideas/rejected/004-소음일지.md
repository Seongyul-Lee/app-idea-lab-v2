# 소음일지

## 한줄 요약
> 층간소음을 자동 감지·측정·기록하여 법적 증거 수준의 소음 리포트를 생성하는 앱

## 해결하려는 문제 (Pain Point)
- **층간소음은 한국 공동주택 거주자의 최대 스트레스 요인**: 환경부 기준 층간소음 민원이 연간 10만 건 이상이며, 전국 아파트 거주민의 78%가 층간소음을 경험 (네이버 블로그 '연세난청센터' 인용)
- **증거 수집이 어려움**: 네이버 지식iN에 "층간소음 증거 어떻게 모으나요"류 질문이 3,300건 이상. 변호사들이 공통으로 "녹음 원본 파일 보존, 파일 생성 일시 기록, 소음일지 작성"을 권고하지만 실제로 이를 체계적으로 하는 도구가 없음
- **기존 데시벨 측정 앱은 증거로 불충분**: 스마트폰 데시벨 앱은 단순 수치만 보여주며, 법적 증거에 필요한 "시간대별 연속 기록, 소음 패턴 분석, 한국 법적 기준 대비 판단"을 제공하지 않음
- **감정적 대응 → 분쟁 악화**: 객관적 데이터 없이 이웃에게 항의하면 갈등만 심화됨. 체계적 기록이 있으면 관리사무소/이웃사이센터/법원에 객관적으로 제시 가능
- **수면 방해, 우울, 스트레스**: 층간소음이 불안·우울을 유발한다는 블로그/뉴스 다수. 경상북도개발공사 공식 블로그에서도 "불안·우울 유발하는 층간소음" 주제로 다룸

## 타겟 사용자
- **주요 타겟**: 아파트/빌라/오피스텔 등 공동주택 거주자 중 층간소음 피해를 겪는 사람
- **연령대**: 20~50대 (자취 청년 ~ 육아 가정)
- **규모 추정**: 한국 공동주택 거주 가구 약 1,200만 가구, 이 중 층간소음 경험률 78% ≈ 약 936만 가구. 실제 적극적 해결을 원하는 층 10~15% 추정 시 약 100~140만 잠재 사용자

## 핵심 기능 (MVP 범위, 최대 3개)
1. **자동 소음 감지 & 데시벨 기록**: 백그라운드에서 마이크로 소음을 모니터링, 설정 임계값 초과 시 자동으로 데시벨 수치·시간·녹음 파일을 기록. 녹음 원본 파일의 생성 일시(타임스탬프)를 변조 불가 형태로 보존
2. **소음일지 자동 생성**: 기록된 데이터를 날짜/시간/데시벨/소음유형별로 정리한 소음일지를 자동 생성. 한국 환경부 '층간소음 기준'(주간 1분 등가소음 39dB, 야간 34dB / 직접충격 주간 57dB, 야간 52dB) 대비 초과 여부를 자동 표시
3. **증거 리포트 PDF 내보내기**: 소음일지 + 녹음 파일 목록 + 데시벨 그래프 + 법적 기준 대비표를 포함한 PDF 리포트 생성. 관리사무소, 이웃사이센터, 환경분쟁조정위원회, 법원 제출용

## 수익 모델
- **프리미엄 구독 (월 3,900원~5,900원)**: 무료 버전은 일 3회 수동 측정 + 기본 소음일지. 프리미엄은 24시간 자동 감지, 무제한 녹음 저장, PDF 리포트 내보내기, 소음 패턴 분석(요일/시간대별 통계)
- **일회성 리포트 구매 (건당 1,900원)**: 구독 없이 특정 기간 리포트만 필요한 사용자를 위한 단건 결제

| 데시벨 측정 앱 (다수) | 단순 실시간 dB 측정만 제공. 자동 감지, 소음일지, 증거 리포트 기능 없음 |

**차별점**: 한국 층간소음 법적 기준을 내장하고, 자동 감지→기록→소음일지→증거 리포트까지 원스톱으로 제공하는 앱은 현재 한국 시장에 없음

## 기술 구현 검토
- **React Native + TypeScript**: 구현 가능. RN의 `expo-av` 또는 `react-native-audio-api`로 마이크 접근 및 오디오 녹음 가능. 백그라운드 녹음은 iOS/Android 네이티브 모듈(Headless Task, Background Fetch) 필요하나 expo-task-manager 등으로 대응 가능
- **Supabase**: 사용자 인증, 소음 기록 데이터(시간, dB, 파일 경로) 저장, PDF 메타데이터 관리에 적합. 녹음 파일 자체는 Supabase Storage에 저장 가능 (무료 1GB, 유료 확장)
- **외부 서비스 의존도**:
  - 결제: RevenueCat(인앱 결제 관리) 또는 RN IAP 직접 구현
  - PDF 생성: `react-native-html-to-pdf` 등 라이브러리로 디바이스 내 생성 가능
  - 별도 외부 API 파트너십 불필요
